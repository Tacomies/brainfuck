# BFC v1.0.0
* Brainfuck compiler

## Building

* Clone the repo
```sh
git clone https://github.com/Tacomies/brainfuck/
```

* Using CMake

```sh
mkdir build
cd build
cmake ..
make
```

## Usage

```sh
bfc <source file path> <destination path>
```

## Dependencies
* [GCC](https://gcc.gnu.org/) is required for BFC to work
